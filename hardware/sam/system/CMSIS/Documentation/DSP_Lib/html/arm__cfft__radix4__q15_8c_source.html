00092 /*  
00093 * Radix-4 FFT algorithm used is :  
00094 *  
00095 * Input real and imaginary data:  
00096 * x(n) = xa + j * ya  
00097 * x(n+N/4 ) = xb + j * yb  
00098 * x(n+N/2 ) = xc + j * yc  
00099 * x(n+3N 4) = xd + j * yd  
00100 *  
00101 *  
00102 * Output real and imaginary data:  
00103 * x(4r) = xa'+ j * ya'  
00104 * x(4r+1) = xb'+ j * yb'  
00105 * x(4r+2) = xc'+ j * yc'  
00106 * x(4r+3) = xd'+ j * yd'  
00107 *  
00108 *  
00109 * Twiddle factors for radix-4 FFT:  
00110 * Wn = co1 + j * (- si1)  
00111 * W2n = co2 + j * (- si2)  
00112 * W3n = co3 + j * (- si3)  
00113   
00114 * The real and imaginary output values for the radix-4 butterfly are  
00115 * xa' = xa + xb + xc + xd  
00116 * ya' = ya + yb + yc + yd  
00117 * xb' = (xa+yb-xc-yd)* co1 + (ya-xb-yc+xd)* (si1)  
00118 * yb' = (ya-xb-yc+xd)* co1 - (xa+yb-xc-yd)* (si1)  
00119 * xc' = (xa-xb+xc-xd)* co2 + (ya-yb+yc-yd)* (si2)  
00120 * yc' = (ya-yb+yc-yd)* co2 - (xa-xb+xc-xd)* (si2)  
00121 * xd' = (xa-yb-xc+yd)* co3 + (ya+xb-yc-xd)* (si3)  
00122 * yd' = (ya+xb-yc-xd)* co3 - (xa-yb-xc+yd)* (si3)  
00123 *  
00124 */
